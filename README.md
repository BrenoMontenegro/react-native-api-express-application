# 📝 Annotation List App

Este é um projeto simples de aplicativo de anotações (bloco de notas), desenvolvido com o objetivo de praticar o desenvolvimento full stack utilizando React Native, Node.js e PostgreSQL.

## Sobre o projeto

A aplicação permite ao usuário:

* Criar novas notas
* Visualizar uma lista de notas
* Deletar notas existentes

## Tecnologias utilizadas

* React Native (Expo)
* Node.js
* Express
* PostgreSQL
* Axios

## Como executar

### 1. Instalar dependências

Backend:

```
cd backend
npm install
```

Mobile:

```
cd mobile
npm install
```

---

### 2. Rodar o backend

```
cd backend
node server.js
```

---

### 3. Rodar o mobile

```
cd mobile
npx expo start ou npm start
```

---

## Objetivo

Este projeto foi desenvolvido com fins educacionais, com o objetivo de reforçar conhecimentos em desenvolvimento full stack mobile.
